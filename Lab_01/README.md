## Дана програма є лабораторною роботою №1 з дисципліни "комп'ютерна графіка"
### Файли в репозиторії запускаються програмою Spyder із пакету Anaconda, який можна завантажити тут: https://repo.anaconda.com/archive/
### Вибирати потрібно найновішу версію x86 для Windows, наприклад: Anaconda3-5.3.0-Windows-x86.exe за посиланням https://repo.anaconda.com/archive/Anaconda3-5.3.0-Windows-x86.exe
